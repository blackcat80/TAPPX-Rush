TEXTBLOB:  !! textblob y nltk trabajan juntas muy bien!

    TextBlob es una biblioteca de procesamiento de lenguaje natural que permite realizar una variedad de tareas de procesamiento de texto, incluyendo análisis de sentimientos, clasificación de texto, extracción de frases clave y corrección ortográfica. La corrección ortográfica de TextBlob se basa en un modelo de lenguaje estadístico y se puede personalizar para incluir palabras específicas.

    Características :
        extracción de frases nominales
        Etiquetado de parte del discurso
        Análisis de los sentimientos
        Clasificación (Naive Bayes, árbol de decisión)
        Tokenización (división de texto en palabras y oraciones)
        Frecuencias de palabras y frases
        análisis
        n-gramos
        Inflexión de palabras (pluralización y singularización) y lematización
        Corrección ortográfica
        Agregar nuevos modelos o idiomas a través de extensiones
        Integración de WordNet

    URL: https://textblob.readthedocs.io/en/dev/

PYSPELLCHECKER:

    PySpellChecker, por otro lado, es una biblioteca especializada en la corrección ortográfica de palabras. Utiliza algoritmos de corrección ortográfica basados en diccionarios para identificar errores ortográficos y sugerir correcciones. PySpellChecker también tiene la capacidad de detectar y corregir errores ortográficos de palabras compuestas y nombres propios.

    En general, si su objetivo es realizar una variedad de tareas de procesamiento de lenguaje natural, TextBlob podría ser una mejor opción, ya que ofrece una amplia gama de funcionalidades. Si su principal necesidad es la corrección ortográfica, PySpellChecker podría ser una mejor opción, ya que está altamente especializado en esa tarea y tiene un enfoque más preciso y detallado en la corrección ortográfica de palabras.

    url: https://www.tutorialspoint.com/python_text_processing/python_spelling_check.htm

SPACY:

    SpaCy es una biblioteca de procesamiento de lenguaje natural (NLP) en Python que se utiliza para realizar tareas de análisis de texto y extracción de información. SpaCy es una biblioteca muy popular en el campo de NLP y se utiliza en muchos proyectos de procesamiento de lenguaje natural.

    SpaCy proporciona una amplia gama de funcionalidades para el procesamiento de texto, que incluyen la tokenización de texto, el etiquetado gramatical, la desambiguación de sentido, la identificación de entidades, el análisis de dependencias, el análisis de sentimientos, el reconocimiento de frases y la generación de respuestas.

    Además, SpaCy está diseñado para ser rápido y eficiente, lo que lo hace adecuado para el procesamiento de grandes volúmenes de texto. SpaCy también se integra bien con otras bibliotecas de Python como pandas, scikit-learn y matplotlib, lo que permite un análisis de texto más completo.

    En resumen, SpaCy es una biblioteca de NLP en Python que permite a los desarrolladores procesar y analizar texto de manera eficiente y efectiva, lo que facilita el desarrollo de aplicaciones basadas en NLP, como la clasificación de texto, la extracción de información y la comprensión del lenguaje natural.

    url: https://spacy.io/

PANDAS: 

    Pandas es una biblioteca de Python para la manipulación y el análisis de datos que proporciona una amplia gama de herramientas para trabajar con datos estructurados y series de tiempo. Pandas es una de las bibliotecas de Python más populares para el análisis de datos y se utiliza en muchos proyectos de ciencia de datos.

    Pandas proporciona estructuras de datos de alto rendimiento y fáciles de usar, como DataFrames y Series, que permiten realizar operaciones de manipulación, agregación y filtrado de datos de manera eficiente. Pandas también proporciona herramientas para la limpieza y transformación de datos, como la eliminación de datos duplicados, la imputación de valores faltantes y el cambio de formato de datos.

    Además, Pandas permite la integración con otras bibliotecas de Python como NumPy y Matplotlib, lo que permite una manipulación y visualización de datos más complejas.

    En resumen, Pandas es una biblioteca de Python para la manipulación y el análisis de datos que proporciona una amplia gama de herramientas para trabajar con datos estructurados y series de tiempo. Pandas es una herramienta esencial para cualquier persona que trabaje con datos en Python, ya sea para análisis de datos, investigación científica, modelado estadístico o aprendizaje automático.

    URL: https://pandas.pydata.org/
